# Socket-Based Log Aggregation System

## Description
This project uses socket programming to send log messages from multiple clients to a central server.

## How it works
- Server listens for connections
- Client sends log messages
- Server stores logs in a file

## Technologies Used
- Python
- Socket Programming

## Output
Logs are stored in logs.txt
